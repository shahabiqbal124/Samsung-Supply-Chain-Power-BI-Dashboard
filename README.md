# Enterprises-Supply-Chain-Power-BI-Dashboard

📊 Samsung Supply Chain – Power BI Dashboard
⚠️ Disclaimer

The data used in this dashboard is entirely fictional and for demonstration/practice purposes only. It does not represent actual Samsung business data, financials, operations, or any real-world figures. This project is not affiliated with or endorsed by Samsung Electronics.


Overview
The Samsung Supply Chain Overview Dashboard is an interactive Power BI report designed to provide end-to-end visibility into a simulated supply chain operation. It consolidates key metrics across Shipment, Inventory, Supplier, and Customer domains into a single, executive-friendly view.

🗂️ Dashboard Pages
PageDescriptionOverviewHigh-level KPIs and summary charts for the entire supply chainShipmentDetailed shipment volumes and carrier-level breakdownInventoryInventory quantity trends and stock distributionSupplierSupplier performance and delivery metricsCustomerCustomer order data and fulfillment insights

Each section is accessible via navigation buttons at the top of the Overview page.

The dashboard is built on imaginary/simulated data structured across the following domains:

Shipment Data – Fictional carrier transactions, shipment quantities, lead times
Inventory Data – Simulated stock levels per product and warehouse
Supplier Data – Mock supplier delivery performance and records
Customer Data – Sample order volumes, fulfillment rates, customer segments
